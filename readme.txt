Add a Plaxo Badge to your WordPress blog. You can display your Plaxo profile info on your Plaxo badge.

== Description ==
Plaxo Badge displays your Plaxo Profile information. To setup and configure your Plaxo Badge, please visit http://www.plaxo.com/pulse/settings/badge

== Installation ==
1. Upzip plaxo-badge.zip
2. Upload 'plaxo-badge.php' to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Go to the Design --> Widgets tab and add Plaxo Badge to your sidebar
5. Click on 'Edit' Plaxo Badge and follow the simple instructions
